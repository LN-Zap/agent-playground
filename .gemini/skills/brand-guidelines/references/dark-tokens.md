# Strike Dark Mode Tokens

Complete token definitions for Strike's dark mode design system.

| Token | Hex | Opacity | Description |
|-------|-----|---------|-------------|
| `Layer/background` | `#000000` | 100% | Root background (OLED black) |
| `Layer/background-hover` | `#1E1E1E` | 100% | Background hover state |
| `Layer/background-pressed` | `#2E2E2E` | 32% | Background pressed state |
| `Layer/background-disabled` | `#2E2E2E` | 32% | Background disabled state |
| `Layer/primary` | `#1E1E1E` | 100% | Primary layer (cards, tiles) |
| `Layer/primary-hover` | `#2E2E2E` | 100% | Primary hover |
| `Layer/primary-pressed` | `#2E2E2E` | 32% | Primary pressed |
| `Layer/primary-disabled` | `#2E2E2E` | 32% | Primary disabled |
| `Layer/secondary` | `#2E2E2E` | 100% | Secondary layer (elevated) |
| `Layer/secondary-hover` | `#3E3E3E` | 100% | Secondary hover |
| `Layer/secondary-pressed` | `#2E2E2E` | 32% | Secondary pressed |
| `Layer/secondary-disabled` | `#2E2E2E` | 32% | Secondary disabled |
| `Layer/tertiary` | `#3E3E3E` | 100% | Tertiary layer (highest) |
